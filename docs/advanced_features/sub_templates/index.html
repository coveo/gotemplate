Defining templates

Razor

@-define("razorTemplate")
  This is a template with a variable here: @.var1  
  For each item in var2:  
  @-for ($item := .var2)
    Print it: @$item  
  @-end for
@-end define

Gotemplate

{{- define "goTemplate" }}
  This is a template with a variable here: {{ get . "var1" }}
  For each item in var2:
  {{- range $item := .var2 }}
    Print it: {{ $item }}
  {{- end }}
{{- end }}

@template("razorTemplate", values){{ template "goTemplate" .values }}
